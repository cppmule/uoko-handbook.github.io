# 服务器规范

服务器规范，方便所有开发人员能够自行发布服务，或者安装需要的插件

* 服务器目录
 
     服务器目录，在所有环境下，都创建了 /data/app-service , /data/micro-service-component,
     /data/software 三个目录。
     
     * /data/app-service 
     此目录用来存放发布的应用服务。
     
     * /data/micro-service-component
     此目录用来存放所有的基础组件目录。
     
     * /data/software
     此目录用来存放所有的安装包。
